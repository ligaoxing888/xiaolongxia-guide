## 你需要准备什么

- **Mac 电脑**（M 系列芯片推荐，也支持 Intel Mac）
- **科学上网工具**（需要稳定连接）
- **一个邮箱**（注册 GPT 账号用，Gmail 推荐）
- **时间：约 30–60 分钟**

## 第一步：注册免费 GPT 账号

1. 打开 https://chatgpt.com
2. 点击 **Sign up**，用邮箱注册
3. 完成验证，登录进去

**不需要付费，不需要绑卡，免费账号就够。**

---

## 第二步：安装 Codex

有两种方式，选一种：

### 方式 A：下载 Codex App（推荐，最简单）

1. 打开 https://developers.openai.com/codex/quickstart
2. 下载 macOS 版 Codex App
3. 安装后打开，用你的免费 ChatGPT 账号登录

### 方式 B：用命令行安装 Codex CLI

打开终端（Command + 空格，输入"终端"），依次执行：

```bash
# 先装 Node.js（如果没有的话）
# 下载地址：https://nodejs.org/en/download/

# 安装 Codex CLI
npm install -g @openai/codex
```

安装完成后，在终端输入 `codex` 启动，用 ChatGPT 账号登录。

---

## 第三步：让 Codex 帮你装小龙虾

打开 Codex，直接说：

> 我要在这台 Mac 上安装 OpenClaw（小龙虾）。安装命令是：`curl -fsSL https://openclaw.ai/install.sh | bash`。请帮我完成完整的安装和配置。

然后 Codex 会开始执行。**遇到需要确认的地方，点 Yes / Approve 就行。**

整个过程就是不停地确认，Codex 会自动处理：
- 检查和安装必要依赖（Node.js 等）
- 下载并安装 OpenClaw
- 完成基础配置

> 💡 遇到报错不要慌，直接把报错信息发给 Codex，让它帮你解决。

---

## 第四步：配置模型

小龙虾装好后，需要接入一个 AI 模型才能对话。

**免费方案：继续用 GPT**

告诉 Codex：
> 帮我配置 OpenClaw，接入 OpenAI 的模型。我用的是免费 ChatGPT 账号。

**进阶方案：接入 Claude（付费，更强）**

如果你后续愿意付费，可以参考我之前的教程，通过 Sub2API 接入 Claude Opus。这是目前思考质量最强的模型。

---

## 第五步：创建 Telegram Bot 并接入

1. 下载 Telegram 桌面版：https://desktop.telegram.org/
2. 在 Telegram 里找 **@BotFather**
3. 发送 `/newbot`，按提示创建机器人
4. 复制返回的 **Bot Token**
5. 回到 Codex，告诉它：

> 帮我把这个 Telegram Bot token 配置到 OpenClaw 里：`你的token`

6. Codex 帮你配好后，去 Telegram 跟你的机器人对话，发一条消息试试

**能回复了，就搞定了。🎉**

## 常见问题

**Q：免费额度用完了怎么办？**
A：Codex 的免费额度主要用来**安装**。装好之后小龙虾是独立运行的，日常对话消耗的是你接入的模型的额度，和 Codex 没关系了。

**Q：限时免费什么时候结束？**
A：OpenAI 没给具体日期，只说 "for a limited time"。建议趁早动手。

**Q：我是 Windows 用户怎么办？**
A：Codex CLI 支持 Windows。流程类似，遇到问题直接问 Codex。

**Q：之后想换成 Claude 模型怎么办？**
A：随时可以。小龙虾的模型配置是可以改的。订阅 Claude 后修改配置文件就行，或者让 Codex 帮你改。

**Q：GPT-5.3 和 Claude Opus 差别大吗？**
A：Claude Opus 在深度思考、长文写作、复杂推理上更强。GPT-5.3 日常用也很好。先用起来，体验差异后再决定。
