## 先回答一个问题：小龙虾是什么？跟 ChatGPT 有什么区别？

这个问题很重要，不搞清楚就容易走弯路。

**ChatGPT / 豆包 / 文心一言……** 这些是"AI 聊天工具"。你问一句，它答一句。关掉窗口，它就忘了你是谁。

**小龙虾（OpenClaw）** 不一样。它是一个 **AI 员工管理系统**。

区别在哪？打个比方：

| | 聊天工具 | 小龙虾 |
|---|---|---|
| 比喻 | 路边问了个聪明人 | 招了个全职员工 |
| 记忆 | 关掉就忘 | 永远记得你说过什么 |
| 主动性 | 你问它才答 | 它会主动提醒你 |
| 多任务 | 一个窗口聊所有事 | 不同岗位不同窗口 |
| 自主性 | 只回答问题 | 能自己操作电脑、读文件、上网查资料 |

**一句话总结：ChatGPT 是顾问，小龙虾是员工。**

## ⚠️ 开始之前，必须看这一段

**这套方案有门槛。我不想骗你说"人人都能搞"——不是的。请先确认你具备以下条件：**

### 1️⃣ 梯子（必须有，而且要稳定）

小龙虾用的是 Claude 模型，Claude 是美国公司 Anthropic 做的，国内直连用不了。

- 推荐美国或日本节点
- **必须开全局 + Tun 模式**（不是只开浏览器代理，是让整台电脑所有流量都走代理）
- 节点不稳定的话，注册容易失败，用着用着也会断

如果你不知道"全局 + Tun 模式"是什么——打开你的梯子软件，找设置，一般会有"系统代理模式"或"Tun模式"的开关，全部打开就行。

### 2️⃣ 海外手机号

注册 Claude 账号需要海外手机号验证，国内 +86 手机号不行。

解决方案：
- **接码平台**（搜"SMS activate"或"5sim"，几块钱买一个一次性号码就行）
- 找海外朋友帮你收一条验证码

### 3️⃣ 支付方式

Claude 订阅需要付费。我用的是 Max 套餐，$125/月（约 950 元）。

**推荐支付方式（按安全性排序）：**

1. **Apple Store 充值 → Claude App 内订阅（最推荐）**
   - iPhone 下载 Claude App
   - Apple ID 充值足够金额
   - 在 App 内订阅，走苹果支付
   - 这是目前最不容易被封的方式

2. **海外信用卡直接支付**
   - 如果有 Visa/Mastercard 海外卡也可以

3. ⚠️ **不推荐虚拟卡 / U 卡**
   - 最近大面积被封号退款，风险极高

### 🚨 重要风险提示：Claude 封号问题

这是我必须跟你说清楚的——最近 Anthropic 对国内用户的风控很严格：

- **虚拟卡、U 卡**订阅的账号，大面积被强制封号退款
- 梯子节点**频繁切换**容易触发风控
- 新号注册后**几天内就被封**的情况不少
- 封号后**申诉成功率很低**

**我自己的降风险经验：**
- 全程**固定同一个节点**，注册和使用都用同一个
- 用 **Apple Store 订阅**
- 注册用 **Gmail 邮箱**
- 登录后**不要频繁切换设备**

## 为什么选这条路，不选更简单的？

我知道市面上有很多"AI 助手"工具，号称零门槛、一键使用。我为什么不用那些？

**第一，不用套壳工具。**

套壳工具就是别人帮你包了一层壳，你用的其实还是 ChatGPT 或 Claude，只是多了一个中间商。

问题在于——**你的天花板，被中间商决定了。** 它不开放的功能，你永远用不了。它涨价了，你只能跟着涨。它倒闭了，你的数据就没了。

搞投资的我，特别讨厌被"套牢"的感觉。**一开始就用套壳，这辈子都要被别人套着。**

**第二，本地安装，不用云端服务器。**

云端服务器意味着你的数据放在别人那里。我的基因检测报告、投资记录、健康数据，这些我不想放在第三方服务器上。

本地安装还有个好处——**文件随手可用。** 基因报告存在电脑桌面，小龙虾直接读取分析，不用先上传到什么平台。

**第三，选 Claude 而不是 ChatGPT。**

这是实际使用后的判断。Claude（尤其是 Opus 版本）的**思考质量**明显更高。同样一个问题，ChatGPT 给你一个"正确但普通"的回答，Claude 给你一个"不仅正确，还能启发你下一步思考"的回答。

我做投资、做社群、做内容——我需要的不是一个只会回答问题的工具，我需要一个**能跟我一起想问题**的伙伴。

**第四，选 Telegram 作为对话入口。**

试过飞书、微信、Web 界面，最后选了 Telegram。原因很简单：

- 不限消息长度
- 支持多个对话窗口（一个群开多个话题，每个话题一个独立的bot）
- 不会被微信审查干扰
- 桌面端和手机端同步
- 支持机器人，配置最简单

## 第一步：安装 Node.js（10 分钟）

**这是什么？** 可以理解为"发动机"。后面所有工具都需要它才能运行。不需要理解它是什么，装上就行。

**怎么装？**

1. 打开浏览器，访问：https://nodejs.org/en/download/
2. 下载 **LTS 版本**（页面上最显眼的那个绿色按钮）
3. 双击安装包，一路"下一步"
4. 安装完成

**怎么验证装好了？**

打开终端（不知道怎么打开？按 **Command + 空格**，输入"终端"，回车），输入：

```bash
node --version
```

看到类似 `v22.x.x` 的版本号，就说明装好了。

> 💡 如果你连终端都没打开过——没关系，我第一次打开的时候也懵了。那个黑乎乎的窗口就是终端，你在里面输入命令，电脑就会执行。把它想象成"跟电脑打字聊天"就行。

## 第二步：安装 Claude Code（15 分钟）

**这是什么？** Claude Code 是 Anthropic 官方出的编程工具。但我们不是用它来写代码——我们用它来**帮我们安装小龙虾**。

它就是你的"装修队长"：你告诉它你想装什么，它替你敲所有命令。

**怎么装？**

推荐先看这个 3 分钟视频（YouTube），跟着做就行：
https://www.youtube.com/watch?v=Eq7VqmZCLQw

或者直接在终端输入：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

等几分钟，它会自动下载安装。

**怎么验证装好了？**

在终端输入：

```bash
claude
```

看到 Claude Code 的启动界面，就成功了。

> 💡 **遇到报错怎么办？** 别慌。把报错信息截图，发给你的备用 AI（ChatGPT / 豆包都行），问它"这个报错怎么解决"。90% 的问题都能这样解决。

Claude Code 需要 Claude 账号才能用。

**充值步骤：**

1. iPhone 上下载 **Claude App**
2. 用 **Gmail 邮箱**注册账号（不要用国内邮箱）
3. 打开 App，点左下角头像

**为什么推荐 Max？**

因为 Max 套餐使用的是 **Claude Opus** 模型——目前最强的 AI 模型之一。思考深度、理解能力、创造力都是 Pro 版本用的 Sonnet 模型比不了的。

我用 Max 账号，完全够用。

## 第四步：安装小龙虾（20 分钟）

**这是最关键的一步，也是最轻松的一步。**

为什么轻松？因为你有了 Claude Code 这个"装修队长"。你不需要自己敲任何安装命令——你只需要跟它说话。

**打开终端，启动 Claude Code：**

```bash
claude
```

然后直接跟它说：

> 我要在这台 Mac 上安装 OpenClaw（小龙虾）。安装命令是 `curl -fsSL https://openclaw.ai/install.sh | bash`。请帮我完成安装和初始配置。

Claude Code 会自动执行安装命令，遇到需要你确认的地方会问你。

**这就是这套方案的核心理念：用 AI 装 AI。**

你不需要懂代码。你不需要查文档。你只需要**会说话**。

> 💡 安装过程中 Claude Code 可能会问你一些问题（比如要不要安装某个依赖）。如果你不确定怎么回答，就把它问的问题截图发给备用 AI，让备用 AI 帮你判断。

## 第五步：配置模型中转——Sub2API（30 分钟）

**这是什么？为什么需要它？**

小龙虾需要一个模型接口才能"思考"。直接用 Claude 官方 API 也行，但 Sub2API 方案更稳定、更省心。

Sub2API 是一个开源工具，它的作用是——**把你的 Claude Max 订阅，变成一个本地 API 接口，让小龙虾直接调用。**

**怎么配置？**

同样交给 Claude Code。把以下内容发给它：

> 请帮我配置 Sub2API，参考这个项目：https://github.com/Wei-Shaw/sub2api
> 以及这篇教程：https://mp.weixin.qq.com/s/s7xzFbnaxE_BVZB2vLgvcA
> 我想让我的 OpenClaw 通过 Sub2API 接入 Claude Opus 模型。

Claude Code 会自动读懂文档和教程，帮你完成部署和配置。

**配置完成后验证：**

在终端输入：

```bash
curl http://localhost:8080/v1/models
```

如果看到返回了模型列表（包含 claude 相关的模型名），就说明 Sub2API 跑起来了。

> 💡 **这一步是整个流程中最容易出问题的。** 如果卡住了，把报错信息发给 Claude Code，让它帮你排查。实在搞不定，也可以跳过这一步，先用 Claude 官方 API（在 claude.ai 后台申请 API Key），后面再慢慢换成 Sub2API。

## 第六步：配置 Telegram 机器人（20 分钟）

小龙虾装好了，现在需要一个"入口"来跟它对话。我选的是 Telegram。

### 6.1 下载 Telegram

- 电脑端：https://desktop.telegram.org/
- 手机端：App Store 搜索 "Telegram"

### 6.2 创建机器人

1. 在 Telegram 搜索 **@BotFather**（注意是带蓝色认证标的那个）
2. 发送 `/newbot`
3. 按提示给机器人起个名字
4. BotFather 会给你一串 **Bot Token**（类似 `123456:ABC-DEF...` 的格式）
5. **把这个 Token 保存好，后面要用**
6. 继续给 BotFather 发：`/setprivacy`
7. 选你刚创建的 bot
8. 选 **Disable**（这样 bot 才能看到群里的消息）

### 6.3 配置 OpenClaw 连接 Telegram

把 Bot Token 告诉 Claude Code：

> 我的 Telegram Bot Token 是 [你的token]，请帮我配置 OpenClaw 连接 Telegram。

Claude Code 会修改配置文件并启动连接。

### 6.4 配对

配置完成后：

1. 在 Telegram 里找到你的 bot，私聊发一条消息
2. 回到终端，执行：

```bash
openclaw pairing list telegram
openclaw pairing approve telegram <配对码>
```

（配对码会在列表里显示出来）

配对成功后，你在 Telegram 里发消息，小龙虾就会回复了。🎉

## 常见问题

**Q：零基础真的能搞定吗？**

能。我就是活生生的例子。中间遇到十几次报错，全部截图扔给 Claude Code 让它解决。真正需要你做的只有三件事：打开终端、复制粘贴、说人话。

**Q：Windows 能装吗？**

能。OpenClaw 支持 Mac、Linux 和 Windows。但 Claude Code 在 Windows 上需要用 WSL（Windows 的 Linux 子系统）。如果你是 Windows 用户，直接问 Claude Code"我是 Windows 系统，帮我安装 OpenClaw"，它会引导你。

**Q：梯子有什么要求？**

关键是**稳定**，以及必须开**全局 + Tun 模式**。不稳定的节点会导致 Claude 连接中断、注册失败。

**Q：遇到看不懂的报错怎么办？**

截图或复制报错文字，直接发给 Claude Code（或者备用 AI）。这是最快的方式。我踩过的所有坑，都是这样解决的。

**Q：为什么不直接用套壳工具？**

套壳工具的天花板，是厂商给你划定的。用原版，你的上限是你自己的想象力。

我现在的 AI 助手，能帮我读基因报告、管公众号排版、定时推送健康日报——**这些功能没有任何一个套壳工具能同时做到。**

**Q：一个月 950 块，值吗？**

我的算法很简单：一个 24 小时在线的 AI 员工，每月 950 元，每天 31 元。

它不迟到、不请假、不抱怨、不需要团建、不需要交社保。

当然，它不会主动帮你想方向——**你才是那个做决策的人。**

**Q：Mac 需要什么配置？**

M 系列或 Intel 都可以，8GB 内存就够。小龙虾本身很轻量，主要的计算都在云端（Claude 的服务器）完成。

**Q：手机上能用吗？**

可以。虽然小龙虾跑在电脑上，但你通过 Telegram 跟它对话——Telegram 手机端和电脑端是同步的。所以你出门在外，掏出手机打开 Telegram，随时能跟你的 bot 们对话。

**Q：多久学会？**

安装本身 3 小时。但"会用"和"用好"是两回事。

我的经验：第 1 天会基础对话，第 2 天开始搭工作流（公众号发文全链路），第 3 天开始探索更多可能性。

**核心技能不是技术，是你有没有想清楚——你到底想让 AI 帮你干什么。**
